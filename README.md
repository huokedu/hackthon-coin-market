# SOV Hackthon #2 : crypto coin market realtime demo (加密貨幣市場價格即時查詢)

A cryto coin market demo by reasonml / reasonml-react.

### Feature
-[] api example (fetch, promise)
-[] json decode example
-[] css / style example
-[] side effect / state management
-[] third party js interop (for chart)
